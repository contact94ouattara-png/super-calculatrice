# 🧮 Super Calculatrice

Calculatrice web interactive développée en **JavaScript vanilla**, **HTML** et **CSS**, permettant d'effectuer les opérations arithmétiques courantes avec une interface simple et intuitive.

## ✨ Fonctionnalités

- Addition, soustraction, multiplication, division
- Calcul de pourcentage
- Gestion de la virgule décimale
- Suppression du dernier chiffre saisi (touche retour)
- Réinitialisation complète (AC)
- Enchaînement de plusieurs calculs à la suite

## 🛠️ Technologies utilisées

- **HTML5** — structure de l'interface
- **CSS3** — mise en forme et design
- **JavaScript** — logique de calcul et manipulation du DOM (sans framework)

## 🚀 Utilisation

1. Clonez le dépôt :
   ```
   git clone https://github.com/contact94ouattara-png/super-calculatrice.git
   ```
2. Ouvrez le fichier `super_calculatrice.html` dans votre navigateur.
3. C'est prêt à l'emploi, aucune installation requise.

## 📌 Concepts mis en pratique

- Sélection et manipulation d'éléments du DOM
- Gestion d'événements (`addEventListener`)
- Logique conditionnelle et structure `switch`
- Gestion d'état applicatif (premier nombre / second nombre / opérateur)
